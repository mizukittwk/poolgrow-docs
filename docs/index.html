Pool Growについて

Pool Growは、ビリヤードの練習記録とフォーム分析を通じて、あなたの上達を支援するアプリです。

主な機能

  • シュート精度の記録と分析
  • フォーム変更の影響を統計的に分析
  • 練習のヒントを自動生成
  • 詳細なフィルタリング機能

プロプラン

プロプランでは以下の機能が利用できます：

  • 1週間の詳細分析
  • 特定日のデータ分析
  • 分析アドバイス - TIPS
